Apa itu KY-SHIRO API?

Semua Endpoint

Base URL: https://apikyshiro.vercel.app

GET /sms Ambil OTP berdasarkan tanggal ▼

Ambil semua OTP dari semua akun iVAS berdasarkan tanggal dan mode yang dipilih. Mode live tidak butuh parameter date.

Parameter
NamaTipeStatusKeterangan
datestringWajib*Tanggal SMS. Format: DD/MM/YYYY. Tidak wajib jika mode=live.
to_datestringOpsionalTanggal akhir range. Default sama dengan date.
modestringOpsionalSumber SMS: received (default), live, atau both.
Contoh Request
GET /sms?date=08/03/2026&mode=received
Contoh Response
{ "status": "success", "mode": "received", "from_date": "08/03/2026", "to_date": "08/03/2026", "total": 2, "accounts_used": 4, "otp_messages": [ { "range": "IVORY COAST 3878", "phone_number": "2250711220970", "otp_message": "Your OTP is 482910", "source": "received", "account": "email@gmail.com" } ] }
GET /health Cek status login semua akun ▼

Melakukan login ke semua akun iVAS dan mengembalikan status masing-masing akun. Digunakan untuk memantau kesehatan API secara real-time.

Parameter

Tidak ada parameter yang dibutuhkan.

Contoh Request
GET /health
Contoh Response (Online)
{ "status": "ok", "login": "success", "accounts_ok": 3, "accounts_total": 4, "details": [ { "email": "akun1@gmail.com", "login": "success" }, { "email": "akun2@gmail.com", "login": "success" }, { "email": "akun3@gmail.com", "login": "failed" } ] }
GET /accounts List akun terdaftar ▼

Menampilkan daftar akun iVAS yang terdaftar di sistem. Password tidak ditampilkan demi keamanan.

Parameter

Tidak ada parameter yang dibutuhkan.

Contoh Request
GET /accounts
Contoh Response
{ "total": 4, "accounts": [ { "index": 1, "email": "akun1@gmail.com" }, { "index": 2, "email": "akun2@gmail.com" } ] }
GET /test Cek semua range & nomor ▼

Tampilkan semua range aktif beserta nomor-nomor yang terdaftar dari semua akun iVAS pada tanggal tertentu.

Parameter
NamaTipeStatusKeterangan
datestringOpsionalTanggal pengecekan. Format: DD/MM/YYYY. Default: hari ini.
Contoh Request
GET /test?date=08/03/2026
Contoh Response
{ "status": "ok", "date": "08/03/2026", "accounts_ok": 4, "total_ranges": 12, "total_numbers": 48, "ranges": [ { "account": "akun1@gmail.com", "range_name": "IVORY COAST 3878", "range_id": "IVORY_COAST_3878", "total_numbers": 5, "numbers": ["2250711220970", "2250711220971"] } ] }
GET /test/sms Cek OTP untuk 1 nomor spesifik ▼

Ambil isi SMS OTP untuk satu nomor spesifik dari range tertentu. Endpoint ini mencoba semua akun hingga menemukan SMS yang sesuai.

Parameter
NamaTipeStatusKeterangan
rangestringWajibNama range iVAS. Contoh: IVORY COAST 3878
numberstringWajibNomor telepon yang dicari. Contoh: 2250711220970
datestringOpsionalTanggal SMS. Format: DD/MM/YYYY. Default: hari ini.
Contoh Request
GET /test/sms?date=08/03/2026&range=IVORY COAST 3878&number=2250711220970
Contoh Response (Ditemukan)
{ "status": "ok", "otp_found": true, "account": "akun1@gmail.com", "range_name": "IVORY COAST 3878", "number": "2250711220970", "otp_message": "Your OTP code is 482910. Valid 5 mins." }
GET /numbers/test-list Daftar test numbers iVAS ▼

Ambil daftar test numbers yang tersedia di iVAS dari semua akun. Data diambil via DataTables AJAX dari iVAS portal.

Parameter
NamaTipeStatusKeterangan
searchstringOpsionalFilter pencarian nomor atau range.
lengthintegerOpsionalJumlah baris yang dikembalikan. Default: 100.
Contoh Request
GET /numbers/test-list?search=ivory&length=50
GET /numbers/add Tambah nomor baru ke iVAS ▼

Menambahkan nomor baru ke sistem iVAS pada range tertentu. Endpoint ini melakukan POST ke iVAS portal secara otomatis.

Parameter
NamaTipeStatusKeterangan
rangestringWajibNama range iVAS tempat nomor akan ditambahkan.
numberstringWajibNomor telepon yang akan ditambahkan.
Contoh Request
GET /numbers/add?range=IVORY COAST 3878&number=2250711220999
Contoh Response
{ "status": "ok", "success": true, "account": "akun1@gmail.com", "number": "2250711220999", "message": "Number added successfully" }
GET /numbers/delete Hapus nomor dari iVAS ▼

Menghapus (return) nomor dari sistem iVAS. Sistem akan mencari nomor di semua akun dan menghapusnya dari akun yang memilikinya.

Parameter
NamaTipeStatusKeterangan
numberstringWajibNomor telepon yang akan dihapus.
rangestringOpsionalNama range untuk mempercepat pencarian.
Contoh Request
GET /numbers/delete?number=2250711220999&range=IVORY COAST 3878
Contoh Response
{ "status": "ok", "success": true, "account": "akun1@gmail.com", "number": "2250711220999", "range_name": "IVORY COAST 3878", "message": "Number returned successfully" }
GET /cookies/set  /cookies/view  /cookies/clear Manajemen cookies ▼

Endpoint untuk mengelola cookies iVAS secara manual. Berguna untuk inject cookies dari browser saat Cloudflare blocking otomatis login.

Endpoint Detail
EndpointMethodKeterangan
/cookies/setGET / POSTInject cookies dari browser secara manual. Parameter: account (email), cookies (string format key=val;key2=val2)
/cookies/viewGETLihat semua cookies tersimpan dari memory dan file kiki.json
/cookies/clearGETHapus cookies. Parameter opsional: account (email). Tanpa parameter = hapus semua.
Contoh Request - Inject Cookies
GET /cookies/set?account=email@gmail.com&cookies=laravel_session=xxx;XSRF-TOKEN=yyy
GET /debug/ranges-raw  /debug/numbers  /debug/sms Debug endpoints ▼

Endpoint debug untuk melihat raw response dari iVAS. Berguna saat troubleshooting parsing atau koneksi ke iVAS.

Endpoint Detail
EndpointParameterKeterangan
/debug/ranges-rawdateRaw HTML ranges dari iVAS. Menampilkan HTML mentah + hasil parsing untuk debug.
/debug/numbersdate, range wajibCek nomor dari range tertentu beserta raw response HTML.
/debug/smsdate, range, number wajibRaw SMS response untuk nomor tertentu.
Contoh Request
GET /debug/ranges-raw?date=08/03/2026 GET /debug/numbers?date=08/03/2026&range=IVORY COAST 3878 GET /debug/sms?date=08/03/2026&range=IVORY COAST 3878&number=2250711220970
